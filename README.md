# Medieval Dynasty Mod Manager

A simple GUI application for managing [UE4SS](https://github.com/UE4SS-RE/RE-UE4SS) and .pak mods in Unreal Engine games.

## Features

- Manage both UE4SS and .pak mods
- Enable/disable mods with a single click
- Toggle all mods on/off with one button
- Modern dark mode UI
- Simple, intuitive interface
- Remove mods with a single click

## Pre-Installation - UE4SS

> [!NOTE]
> Optional if you intend to manage .pak mods only

1. Download the experimental-latest release of UE4SS from [here](https://github.com/UE4SS-RE/RE-UE4SS/releases)
2. Extract the contents of the zip file into the `Medieval_Dynasty/Binaries/Win64` folder

## Installation

1. Download the latest release from the [Releases page](https://github.com/ConnorC432/ue4ss_modmanager/releases)
2. Place the executable in your game's `Medieval_Dynasty` folder
> [!WARNING]
> Also make sure to copy the `_internal` folder, and it's contents, alongside the executable, if using the Windows build
3. Run the executable

## Usage

1. Launch the application
2. Mods in your `UE4SS/Mods` folder and `Content/Paks` folder are automatically detected
3. Enable/disable mods by checking/unchecking their boxes
4. Use "Toggle All" to enable/disable all mods at once
5. Use "Import Mod" to import mods from an archive
6. Click "Save Changes" to apply your configuration
